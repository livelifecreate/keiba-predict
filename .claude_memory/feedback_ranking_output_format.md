予想ランキングをチャットに出力する際は、以下のフォーマットで**必ず**表示する。

- 全頭表示（上位のみNG）
- ソートは**システムランキング順**（スコア降順）
- 列: システム順位 / 人気 / 馬番 / 馬名 / 単勝オッズ / 複勝(3着払い)

**Why:** 「さっき出した表形式で」と指定があったため。毎回この形式が標準。

**How to apply:** `scorer.py` 実行後のチャット出力時に必ずこの表を添付する。オッズが未取得の場合は「オッズ未取得」と記載して表の枠だけ出す。
